Лопарёв Влад, КНМО-101.
===Постановка задачи:===
На вход получаем сообщение на английском языке.
С помощью code.js кодируем сообщение, указав необходимый сдвиг алфавита.
    -обрабатываем символы, не являющиеся буквами;
    -обрабатываем большие буквы;
    -осуществляем сдвиг;
    -возвращаем большие буквы;
    -записываем закондированное сообщение в файл.
С помощью decode.js декодируем сообщение.
    -задаём массивы канонических и фактических частот символов языка;
    -обрабатываем символы, не являющиеся буквами;
    -обрабатываем большие буквы;
    -высчитываем фактические частоты;
    -циклично, для каждого сдвига, вычисляем значение функции суммы и 
     при нахождении минимума запоминаем сдвиг;
    -осуществляем обратный сдвиг;
    -возвращаем большие буквы;
    -записываем декодированное сообщение в файл.

===Пример:===
Для кодирования:
> node .\code.js .\input_message.txt .\coded_message.txt 13 
                          1                    2         3
1 - исходное сообщение;
2 - куда сохранить закодированное сообщение;
3 - сдвиг.

Для декодирования:
> node .\decode.js .\coded_message.txt .\decoded_message.txt
                            1                    2
1 - закодированное сообщение;
2 - куда сохранить декодированное сообщение.
