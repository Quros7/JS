Постановка задачи:
Дано арифметическое выражение.
— Необходимо преобразовать его к обратной польской записи.
— По полученной строке вычислить значение выражения.
— Сравнить результаты работы алгоритма и функции eval().

Реализовал программу, которая обрабатывает выражения,
состоящие из однозначных целых неотрицательных чисел.
Поддерживает операции +, -, *, /, ^.
Обработано использование некорректных операторов.
